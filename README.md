
#[Predawn](http://jamiewilson.io/predawn) for [HyperTerm](https://hyperterm.org/)

![Screenshot of Predawn for Hyperterm](http://i.imgur.com/crOzaqr.png)

## Installation

Add to the `plugins` array in your `.hyperterm.js` file:

```js
module.exports = {

  config: { /*... */ },

  plugins: [
    "predawn-hyperterm"
  ]

};
```
